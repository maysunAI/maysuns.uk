# Python 基础

## 它是什么

Python是一种通用编程语言，以可读性强、接近纯英语的语法和一个极其庞大的库生态系统而闻名。它常被推荐作为第一门编程语言，因为它的语法不会挡在学习底层概念的路上；同时它也能扩展到严肃的生产环境使用——它不只是一门「初学者语言」。

## 为什么它可读性强

Python用缩进本身来定义代码结构（而不是花括号或明确的代码块标记），并偏好使用一小套清晰、一致的语法结构。结果是代码往往读起来接近它所表达的逻辑本身，这降低了编写它的门槛，也降低了之后别人（或者一个AI助手）理解它的门槛。

## 它牺牲了什么

Python是一门解释型、动态类型的语言，这通常意味着它的原始执行速度比编译型、静态类型的语言（比如C++或Rust）要慢。在实践中，这对大多数应用场景其实并不太重要——而在那些原始速度真正重要的地方，Python代码通常会去调用底层快速的、编译好的库（这正是大多数数值计算/机器学习相关Python代码的工作方式），而不是在纯Python里自己完成繁重的运算。

## 它是默认选择的领域

- **数据相关工作** ——用于数据处理、分析和可视化的库都很成熟，使用极其广泛。
- **机器学习/AI** ——大多数主流机器学习框架都把Python作为其主要接口暴露出来，使它成为训练和使用模型时的默认语言。
- **脚本编写与自动化** ——快速的一次性脚本（重命名这些文件、抓取这份数据、自动化这个重复性任务）是Python最强大的日常用途之一。
- **Web后端** ——像Flask和FastAPI这样的框架（参见相关文章）也让Python成为构建API和Web后端时的常见选择。

## 生态系统：pip

Python的包管理器**pip**提供了对Python Package Index（PyPI）的访问——一个庞大的公共可复用包库，这意味着大多数常见问题已经有了相当成熟的解决方案,而不需要从零开始构建。
